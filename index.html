Ongoing projects

Surgical planning

Create a automatic pipeline to analyse a diffusion magnetic resonant image of brain with tumor. The analysis consist to generate a whole brain tractogram with a tumor segmentation prior and extract the white matter bundles near of the tumor.

State-of-the-art pipeline and QA

Create a Nextflow pipeline with the state-of-the-art processes to analyse human brain database. The goal is to improve the speed and the reproductibility. This project is based on Nextflow and Singularity container.

Improve tractography

Improve the tractography with multi-shell multi-tissue fODF in the macaque visual cortex. The ground truth used is based on tract tracing and histology.
